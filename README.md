Udemy Clone Website (HTML & CSS)

This project is a Udemy Clone Website built to replicate the look and feel of the Udemy online learning platform. The main goal of this project is to practice HTML and CSS by creating a real-world styled e-learning website interface.

The website includes a structured homepage with a navigation bar, search bar UI, course categories, recommended courses, popular courses, and a detailed footer section similar to the original Udemy website.
